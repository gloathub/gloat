Gloat Documentation
===================

[Gloat](../index.md) compiles [Clojure](https://clojure.org) and
[YAMLScript](https://yamlscript.org) to Go source, native binaries,
WebAssembly, and shared libraries; with no JVM at runtime and nothing to
install globally.

The pages below are the canonical reference docs.
Each one is also installed as a man page, so `man gloat-repl` opens the
same content as the web page.

- [Getting Started](getting-started.md) - first steps with Gloat.
- [Installation](gloat-install.md) - installation and platform notes.
- [REPL Reference](gloat-repl.md) - the Glojure REPL in detail.
- [Go Interop](gloat-go-interop.md) - calling Go from Glojure code.
- [Java Interop](gloat-java-interop.md) - interoperating with the JVM.
- [Tutorial](gloat-tutorial.md) - tour of the gloat ecosystem.

New to Gloat?
The [home page](../index.md) and the [Demo](../demo.md) are the best starting points.
